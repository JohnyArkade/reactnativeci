## How to Install
```bash
# clone this repository
 git clone https://github.com/subekti404dev/todos-app-iman-rn.git

# install dependencies
cd todos-app-iman-rn && yarn install

# run on android emulator
yarn android

# run on ios simulator
yarn ios

```
